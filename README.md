# Lakshya AI Trading Assistant

## Setup Instructions:

1. **Gemini API Key Lena:**
   - Jao: https://makersuite.google.com/app/apikey
   - Create API Key
   - Copy karo

2. **Code Mein Paste Karna:**
   - `index.html` kholna
   - Line 319 pe `YOUR_GEMINI_API_KEY` ki jagah apna key daalo

3. **GitHub Pe Upload Karna:**
   - GitHub account banao
   - New repository: `trading-assistant`
   - Files upload karo
   - Settings → Pages → Enable

4. **Open Karna:**
   - `https://yourusername.github.io/trading-assistant`

## Features:
- ✅ Voice commands (Hindi)
- ✅ Auto screenshot analysis
- ✅ Live chart (TradingView)
- ✅ Signal generation
- ✅ News feed
- ✅ 100% Free

## Limitations:
- Browser open rehna chahiye
- Real data ke liye API key chahiye
- 60 requests/minute (Gemini limit)
